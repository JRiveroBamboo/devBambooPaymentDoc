Central America

Receive payments in Costa Rica, El Salvador, and Nicaragua using PuntoXpress, Central America’s most extensive cash collection network with regional coverage.

This section shows the available payment methods, currencies, and related information to consider when processing in Costa Rica, El Salvador, and Nicaragua.

Payment methods

Payment MediaId Payment Method Purchase Pre-authorization Full refund Partial Refund Type Flow
28 PuntoXpress Cash API

Currencies

Code Description Mode
USD US Dollar Domestic* and CrossBorder
CRC Costa Rican colón Domestic and CrossBorder
NIO Nicaraguan córdoba Domestic and CrossBorder

*USD for domestic purchases is only available for El Salvador.
